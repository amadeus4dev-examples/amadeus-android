        FIND HOTELS. This service returns a list of hotels with multiple shopping offers around a city center or geolocation for a given date range. It is also possible to directly ask for a specific list of hotels. The user can refine the search criteria by asking for specific hotel categories, hotel chains or hotel facilities, such as parking or restaurant. It is also possible to ask for a specific meal or budget. By default, only available hotels are returned, with the cheapest available option for each hotel. It is however also possible to retrieve details about the closed properties, and to retrieve all available offers per hotel. Warning: at this step only preliminary prices are returned. Use the next service to ask for a single hotel to retrieve exact prices. Minimum mandatory parameters for a hotel search (mutually exclusive): - cityCode - geocodes (longitude and latitude) - list of hotels ### Query example Get the cheapest hotel offers in Paris today with a search radius of 5 Km around the city center.         bestRateOnly - use to return only the cheapest offer per hotel or all available offers (optional, default to true)

        pageLimit - maximum number of hotels to return in each response (optional)
