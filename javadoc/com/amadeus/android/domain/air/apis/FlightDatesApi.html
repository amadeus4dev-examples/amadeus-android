        Parameters:
        origin - IATA code of the city from which the flight will depart IATA table codes - e.g.
        destination - IATA code of the city to which the flight is going.
        departureDate - the date, or range of dates, on which the flight will depart from the origin.
        oneWay - if this parameter is set to true, only one-way flights are considered.
        duration - exact duration or range of durations of the travel, in days.
        nonStop - if this parameter is set to true, only flights going from the origin to the destination with no stop in-between are considered (optional, default to false)
        maxPrice - defines the price limit for each offer returned.
        viewBy - view the flight dates by DATE, DURATION, or WEEK.
